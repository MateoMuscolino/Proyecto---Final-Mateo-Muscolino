# Proyecto-Final-Muscolino-Mateo #

# Instrucciones instalar proyecto en local
- Crea una carpeta contenedora base
- Abre la consola y ubicate en la carpeta base
- Clona este proyecto en la carpeta base
- Entra en la carpeta que acabas de clonar
- Para instalar las dependencias ejecuta este comando:
-------------------------------
pip install -r requirements.txt
-------------------------------
